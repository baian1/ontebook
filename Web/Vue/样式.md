# class
```
//基础
v-bind:class="{ active: isActive, 'text-danger': hasError }"
//用对象表示
v-bind:class="classObject"
data: {
  classObject: {
    active: true,
    'text-danger': false
  }
}
```
```
//数组语法
v-bind:class="[activeClass, errorClass]"
data: {
  activeClass: 'active',
  errorClass: 'text-danger'
}
```
# 内联
```
v-bind:style="{ color: activeColor, fontSize: fontSize + 'px' }"
data: {
  activeColor: 'red',
  fontSize: 30
}
```
