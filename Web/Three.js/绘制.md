# three 绘制

1. projectObject遍历scene内的物体
    - light 添加到renderstate中的light中
    - transparent 添加到currentRenderList的transparent
    - op 添加到currentRenderList的opaque
2. 如果有设置排序,默认排序: painterSortStable
    1. groupOrder
    2. renderOrder
    3. program.id
    4. material.id
    5. a.z
    6. a.id
    如果是透明物体,反向排序:reversePainterSortStable
    先绘制远处再绘制近处
