# 简单数据类型
1. 可以使用typeof来判断
```
typeof 'abc'  // "string"
typeof false  // "boolean"
typeof 123  // "number"
typeof undefined  // "undefined"
typeof Symbol(5)  // "symbol"
typeof function(){}  //"function"
```
null属于对象

2. 可以通过类型上的construct来进行判断,原理是基本类型读取constructor时会创建一个包装类型
