# 数字分组

```
//之前的数字
10000000000
1019436871.42

//现在可以对其分组
1_000_000_000
1_019_436_871.42
```

# Bigint

使用 n 后缀

## 格式化

```
const nf=new Intl.NumberFormat('fr');
nf.format(12_345_123n);
---->
12 345 123
```

## 存储

在内存中创建一个对象,保存一系列的块,存储着数据

```
{
  type: 'BigInt',
  sign: 0,
  num_digits: 3,
  digits: [0x12…, 0x34…, 0x56…],
}
```
