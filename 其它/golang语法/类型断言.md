# 类型断言
```
var i interface{} = "hello"

//返回hello
s := i.(string)
fmt.Println(s)

//返回hello和true
s, ok := i.(string)
fmt.Println(s, ok)
```

# 类型选择
```
switch v := i.(type) {
case T:
    // v 的类型为 T
case S:
    // v 的类型为 S
default:
    // 没有匹配，v 与 i 的类型相同
}
```

# Stringer接口
fmt 包（还有很多包）都通过此接口来打印值。
```
type Person struct {
	Name string
	Age  int
}

func (p Person) String() string {
	return fmt.Sprintf("%v (%v years)", p.Name, p.Age)
}

a := Person{"Arthur Dent", 42}

//将触发String()
fmt.Println(a, z)
```
