# 接口

```go
type Abser interface {
  Abs() float64
}


//实现
type MyFloat float64

func (f MyFloat) Abs() float64 {
  if f < 0 {
    return float64(-f)
  }
  return float64(f)
}

f := MyFloat(-math.Sqrt2)

var a Abser
a = f  // a MyFloat 实现了 Abser
```

## 隐式实现

```go
type I interface {
  M()
}

type T struct {
  S string
}

// 此方法表示类型 T 实现了接口 I，但我们无需显式声明此事。
func (t *T) M() {
  fmt.Println(t.S)
}
```

## 底层值为 nul 的接口

```go
type I interface {
  M()
}

type T struct {
  S string
}

func (t *T) M() {
  if t == nil {
    fmt.Println("<nil>")
    return
  }
  fmt.Println(t.S)
}

var i I
var t *T
i = t

describe(i)
//(<nil>, *main.T)
i.M()
//<nil>
```

## 空接口

interface{}  
可以用来保存任何值

```go
var i interface{}
describe(i)

i = 42
describe(i)

i = "hello"
describe(i)
```
